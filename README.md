# MIT 1.001 Term Project

A term project for MIT 1.001: Engineering Computation and Data Science, taught by Abel Sanchez and John R. Williams.

## Overview

LightMap — a real-time web map of shade during the day and lighting at night in Boston and Cambridge, Massachusetts, computed from sun position, building geometry, streetlights, tree canopy, and weather. The project began as a healthcare AI systems exploration and arrived at LightMap after faculty feedback pointed toward a more intuitive, map-based direction.

Phase 5 submission readiness plan (video storyboard, priority tasks) drafted.
