# MIT 1.001 Term Project

Term project for MIT 1.001 (Abel Sanchez, John R. Williams).

LightMap — a real-time web map of shade during the day and lighting at night across Boston and Cambridge, Massachusetts. Built with FastAPI + MapLibre GL JS. Arrived at from the healthcare AI systems exploration and a faculty pivot toward a more intuitive, map-based direction.

See `docs/` for planning, research, and competitive analysis.
